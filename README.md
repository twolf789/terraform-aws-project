##Prerequisites:
  1. Install Terraform - 
     `brew tap hashicorp/tap`

     `terraform -version`

  2. Install awscli -
     `brew install awscli`
     `aws --version`
  
  3. Install aws-vault -
     `brew install --cask aws-vault`
